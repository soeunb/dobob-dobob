# dobob_icon_pack_from_guide

최종 가이드 이미지에서 그대로 잘라낸 도밥도밥 아이콘팩입니다.

구성:
- png_256: 앱 적용용 고해상도 PNG
- png_64: 작은 리스트/미리보기용 PNG
- svg_wrapped: PNG를 SVG 안에 래핑한 파일(시각적으로 동일하게 보이도록 만든 버전)
- reference: 원본 가이드 이미지

권장 적용:
- 간식 리스트 아이콘: 18~20px
- 카드/추천 아이콘: 24~32px
- Empty state 아이콘: 48~56px

주의:
- 이 파일들은 가이드 이미지에서 추출한 래스터 기반 아이콘입니다.
- 완전한 path 기반 벡터가 아니라, '가이드와 똑같이 보이게' 하기 위한 PNG/SVG wrapper 버전입니다.
